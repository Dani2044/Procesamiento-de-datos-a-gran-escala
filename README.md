# Procesamiento de Datos a Gran Escala

Este repositorio contiene los materiales desarrollados para la asignatura **Procesamiento de Datos a Gran Escala**, incluyendo talleres, proyectos y actividades prácticas orientadas al análisis y manipulación de grandes volúmenes de datos mediante Python y herramientas asociadas.

## Contenido del repositorio

En este repositorio se incluyen:

- Talleres prácticos de Python y procesamiento de datos  
- Proyectos académicos de la asignatura  
- Notebooks Jupyter con ejercicios y ejemplos  
- Scripts y recursos de apoyo para el desarrollo de las actividades  

## Requisitos

El repositorio cuenta con un archivo `requirements.txt` que especifica las dependencias necesarias para ejecutar los notebooks y scripts incluidos. Se recomienda crear un entorno virtual e instalar las dependencias antes de comenzar.

## Uso

Los contenidos están organizados por carpetas según el tipo de actividad.
