# Taller 1 – Introducción a Python

Este directorio contiene los notebooks correspondientes al **Taller 1**, orientado al aprendizaje progresivo de los fundamentos del lenguaje Python.

## Contenido del Taller 1

Los archivos Jupyter Notebook (`.ipynb`) incluidos en este taller son:

- **01-Python-Cadenas.ipynb**  
  Operaciones básicas con cadenas de texto.

- **02-Python-Tuplas.ipynb**  
  Uso y características de las tuplas.

- **03-Python-Listas.ipynb**  
  Manejo de listas y sus métodos principales.

- **04-Python-Conjuntos.ipynb**  
  Conjuntos y operaciones básicas.

- **05-Python-Diccionarios.ipynb**  
  Diccionarios y manejo de pares clave–valor.

- **06-Python-Condiciones.ipynb**  
  Estructuras condicionales.

- **07-Python-Bucles.ipynb**  
  Uso de bucles `for` y `while`.

- **08-Python-Funciones.ipynb**  
  Definición y uso de funciones.

- **09-Python-Clases.ipynb**  
  Introducción a clases y objetos.

- **Practico_Bono_1.ipynb**  
  Actividad práctica de refuerzo del Taller 1.

## Requisitos

- Python 3.x  
- Jupyter Notebook o Visual Studio Code con extensión Jupyter
